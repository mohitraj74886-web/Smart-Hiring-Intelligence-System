# Smart Hiring Intelligence System

## Overview
An end-to-end Data Science project that automates resume screening,
candidate-job matching, and employee attrition prediction using
Machine Learning and SQL.

## Features
- Resume parsing using NLP
- Skill extraction
- TF-IDF + Cosine Similarity matching
- Attrition prediction using ML
- Interactive PowerBi dashboard

## Tech Stack
Python, SQL, Machine Learning, NLP, PowerBi

## Use Case
Helps HR teams reduce hiring time and predict employee attrition.
